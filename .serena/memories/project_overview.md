Purpose: Batch registration utility for Grok/x.ai accounts with optional NSFW/Unhinged post-signup enablement. Architecture is split into a main CLI runner (grok.py), a local Quart-based Turnstile solver service (api_solver.py), supporting service modules under g/, and operational scripts for one-click startup and smoke checks. The codebase is Windows-first but also ships bash helpers.

Tech stack: Python 3.10-3.12, curl_cffi requests impersonation, requests, Quart, Patchright, optional Camoufox, BeautifulSoup, python-dotenv, Rich. Tests use unittest and IsolatedAsyncioTestCase.

Structure: grok.py contains CLI entrypoint, bounded stop policy, JSONL metrics, bootstrap scraping, gRPC-web signup helpers, worker orchestration. api_solver.py exposes /turnstile and /result endpoints and manages a browser pool for solving Cloudflare Turnstile. g/email_service.py handles moemail OpenAPI mailbox lifecycle and verification code extraction. g/turnstile_service.py brokers between YesCaptcha and the local solver. g/user_agreement_service.py and g/nsfw_service.py perform post-signup gRPC/REST actions. db_results.py stores solver results in an in-memory async-safe map. start_all.ps1/start_all.sh/StartAll.bat orchestrate solver + grok end-to-end. release_smoke.ps1 runs unit tests and solver health checks.
